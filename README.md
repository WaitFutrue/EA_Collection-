## 文件上传的「黄金法则」✔

### （1）三不上传原则❌

- **不上传原始工程文件**

Altium Designer 等大型工程 → 只上传 PDF 版原理图 + STEP 模型
或者一些简单的常用模块库，以简为主

- **不上传二进制构建产物**

固件 .bin 文件 → 通过 GitHub Releases 管理

- **不上传临时调试数据**

逻辑分析仪原始数据 → 精选关键波形截图 + 文字分析
